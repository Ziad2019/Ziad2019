### 📌 Featured Project — E-Shop

A complete fullstack e-commerce platform: a modular NestJS backend paired with a Next.js frontend, covering the customer storefront, checkout, and an admin dashboard.

- Multi-strategy authentication with Passport.js (JWT access/refresh tokens with rotation & reuse detection, plus Google OAuth)
- Admin dashboard with full CRUD on products, order management, and user role control
- Stripe payment integration, image optimization with Multer/Sharp
- Swagger/Postman API documentation

🔗 **Live:** [e-shop-frontend-48s8.vercel.app](https://e-shop-frontend-48s8.vercel.app/)
